# Слайдер выбора цвета текста и фона
Слайдер для выбора цвета текста и фона, переключение между настройками текста и фона происходит через переключатель, изменения применяются мгновенно.

## Стек технологий

- **HTML, CSS**
- **JS**
- **JQuery**
